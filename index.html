Lesson 2: Creating a Structured Document with HTML

CSS (Cascading Style Sheet)

Controls style of the page (how it looks).

DOM (Document Object Model)

A cross-platform and language-independent convention for representing and interacting with objects in HTML.
The nodes of every document are organized in a "tree-like structure".

Boxify

When coding webpages, it helps to think of everything as boxes. Programmers "boxify" all contents on the page to make it easier to create structures.

Text Editors

Sublime is an example of a text editor that programmers use to code.

Lesson 3: Adding CSS Style and HTML Structure

CSS

You can identify a class or tag of the element you want to find (called a match) and replace it with different ones.
Inheritance: "Cascading" means rules are applied to the element that matches (ancestor) but also to those element's child elements (descendants).
Styles are applied in the following order:
1. Default style of the browser.
2. Stylesheet in a separate file.
3. Stylesheet inside HTML.
4. Inline style in an element.

Example:
h1 {
background-color : black;
color : white;
}
h1 is a selector.
background-color : black is a declaration.
background-color is a property.
black is the value.

You can select by class as well using .class-name

Box Sizing

1. Set sizes in terms of percentages.
2. Set box-sizing attribute to box-border for every element

Box Positioning

Divs are block elements, however, using display : flex can let divs appear next to each other.
